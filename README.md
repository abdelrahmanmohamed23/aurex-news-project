# Aurex News
A fast, SEO-friendly, and accessible News SPA built with Vanilla JavaScript, featuring seamless browser history and custom JSON data integration.
---
## ✨ Features
- Fully responsive design across all devices.
- Performance-focused architecture.
- SEO-friendly HTML structure.
- Responsive navigation menu.
- Custom client-side router built with the History API.
- News slider.
- Dynamic article pages generated from news data.
- Theme switching (Light/Dark Mode).
---

- HTML
- CSS
- JavaScript
- Vite
